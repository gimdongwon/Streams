### 세줄요약

- "1-10" 까지 1개 "11-20" 까지 2개씩 "21-30" 까지 1개, 조커 1개의 총 41개의 랜덤 숫자를 가짐.
- 랜덤으로 숫자를 20개를 차례대로 선출하여 턴마다 1개씩 숫자를 분배
- 오름차순으로 계산하여 최고 승점을 가진 사람이 승리.
